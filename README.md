# hello-ogre3d

Working sample how to prepare OGRE-based project.

### Environment

* Ubuntu 20
* OGRE 1.12 (build from source)

# How to run

```bash
mkdir -p  build
cd build
cmake ..
make
./hello-ogre3d
```
