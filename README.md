# hello-ogre3d

Working sample how to prepare OGRE-based project.

### Environment

* Ubuntu 20 / Windows 10
* OGRE 1.12 (build from source)

### Requirements

##### Ubuntu

```bash
bash ./install-ogre3d.sh
```

##### Windows

```bash
# git bash
bash ./install-ogre3d-win.sh
```

### How to run

```bash
mkdir -p build
cd build
cmake ..
make
./hello-ogre3d
```
